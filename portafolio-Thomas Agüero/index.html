Mis Proyectos

Página de Emprendimiento - Porcelana Fría e Impresiones 3D

Este es mi primer proyecto personal, actualmente en desarrollo. Se trata de una página web creada para mi propio emprendimiento de porcelana fría e impresiones 3D. El sitio cuenta con una estructura modular similar a esta misma web, con una barra de navegación superior que enlaza a secciones como “Sobre Nosotros”, “Catálogo”, “Enlaces” y “Contacto”. Además, incluye una galería interactiva de productos y enlaces a redes sociales.
